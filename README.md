# Efficient-LLM-Inference
Repository for Deep Learning with Massively Parallel Acceleration project. The goal was to compare several compression and optimization techniques for LLaMA 3.2 models, and focus on trade-offs between memory usage, inference speed, latency, and model quality.

The [main notebook](dlmpa_compression_study.ipynb) contains the whole experimental workflow, including environment setup, as well as documentation and conclusions.  

The benchmark results are stored in [results.csv](results/results.csv), and the generated plots for the notebook are available in the [results folder](results/).
